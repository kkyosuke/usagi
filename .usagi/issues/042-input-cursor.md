# 入力フィールドのカーソル移動対応

## 概要

Session 作成時などの TUI 入力フィールドで、現状は末尾への追記・削除しかできず、文字列の途中を編集できない。カーソルを左右に移動して任意の位置で編集できるようにする。

## やること

- 入力フィールドにカーソル位置の状態を持たせる。
- ←/→ でカーソル移動、Home/End（または Ctrl+A / Ctrl+E）で行頭・行末移動。
- カーソル位置での文字挿入・削除（Backspace / Delete）に対応する。
- カーソル位置を画面上に表示する。
- マルチバイト文字（日本語）でも正しく移動・削除できるようにする。

## 完了条件

- Session 作成入力をはじめ各入力フィールドで、文字列の途中にカーソルを移動して挿入・削除できる。
- 日本語入力でもカーソルがずれない。
