# 前回セッションの継続実行（claude -c 相当）

## 概要

`claude -c`（直前の会話を継続）のように、対象 worktree で前回の Claude セッションの続きから実行できるようにする。中断・離席後に文脈を引き継いで再開したいケースに対応する。

## やること

- Attach / 実行時に「前回の続きから再開する」オプションを用意する。
- 起動コマンドに継続フラグ（`-c` / `--continue` 相当）を付与する。
- 継続可能な過去セッションが存在するかを判定し、なければ通常起動にフォールバックする。

## 完了条件

- 対象 worktree で前回のセッションを継続した状態で Claude を起動できる。
- 過去セッションが存在しない場合は通常起動になる。
