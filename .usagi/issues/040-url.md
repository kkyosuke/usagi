# リンクを開けるようにする

## 概要

Attach 中のターミナル表示などに含まれる URL を、TUI から直接ブラウザで開けるようにする。PR の URL などをすぐ確認したいケースに対応する。

## やること

- 出力テキストから URL を検出する。
- URL を選択・クリック（またはキー操作）でブラウザに渡して開く。
- OS ごとのオープン手段（macOS `open` / Linux `xdg-open` / Windows）を扱う。

## 完了条件

- 画面上の URL を操作するとデフォルトブラウザで開く。
- 複数 OS で動作する。
