## 変更内容

- IPC に `Keys { worktree, data }`（入力バイト列）/ `Resize { worktree, cols, rows }` を追加。
- daemon は該当 worktree の `PtySession` へ `write` / `resize`。応答はなく、結果は既存の `Screen` push で attach クライアントへ流れる。
