# 2. プロジェクト選択画面（Open）

> [画面設計トップ](README.md) ｜ ← 前へ [1. 起動画面（Welcome）](01-welcome.md) ｜ 次へ → [3. 新規プロジェクト画面（New）](03-new.md)

起動画面で `Open`（`o` / Enter）を選ぶと遷移する、登録済みワークスペースの一覧から
開く対象を選ぶ画面。一覧は `workspaces.json` 由来で、最終利用の新しい順（最近開いたものが上）に
並びます。起動画面と同じ代替スクリーン上に描画されます。

> ワークスペースを選択するとホーム画面（[5. ホーム画面](05-home.md)）へ遷移し、そのワークスペースの
> worktree 一覧を表示します。ホーム画面で `Esc` を押すとこのプロジェクト選択画面へ戻ります。

## レイアウト

リスト全体を 1 つのブロック（幅 52）として端末中央に寄せ、ブロック内は左揃えで描画します。

```text
┌───────────────────────────────────────────────────────┐
│                       (\(\                            │  ┐ うさぎ AA（マゼンタ・太字）
│                      (='-')                           │  │
│                      o(_(")(")                        │  ┘
│                    Open Project                       │  ← タイトル（緑・太字）
│              Open a registered workspace              │  ← サブタイトル（淡色）
│                                                       │
│      > my-project    /path/to/my-project              │  ┐ プロジェクト一覧
│        other-repo    /path/to/other-repo              │  ┘  選択行：赤 ">" + 名前シアン太字
│                                                       │     パスは淡色（長い場合は先頭を … で省略）
│                                                       │  ← 通知行（任意・黄色／読み込み失敗時など）
│                                                       │
│        ↑↓: move / Enter: open / Esc: back             │  ← フッター（淡色）
└───────────────────────────────────────────────────────┘
```

ワークスペースが 1 つも登録されていない場合は、一覧の代わりに
`No workspaces yet. Choose "New" to add one.`（淡色）を中央に表示します。

## 構成要素

| 要素 | 内容 | スタイル |
|---|---|---|
| タイトル | `Open Project` | 緑・太字 |
| サブタイトル | `Open a registered workspace` | 淡色（dim） |
| プロジェクト一覧 | 登録済みワークスペースを「カーソル + 名前(幅揃え) + パス」で 1 行ずつ表示。パスはブロック幅に収まるよう先頭を省略 | 選択行：`>` 赤太字／名前 シアン太字。パスは常に淡色 |
| 空表示 | 登録が無い場合のメッセージ | 淡色（dim） |
| 通知行 | 読み込み失敗時などのメッセージ。未表示時も 1 行ぶんの枠を確保しレイアウトを安定させる | 黄色 |
| フッター | 操作ヘルプ `↑↓: move / Enter: open / Esc: back` | 淡色（dim） |

## キー操作

| キー | 動作 |
|---|---|
| `↑` / `k` | 選択を 1 つ上へ移動（先頭で押すと末尾へラップ）。通知をクリア |
| `↓` / `j` | 選択を 1 つ下へ移動（末尾で押すと先頭へラップ）。通知をクリア |
| `Enter` | 選択中のワークスペースを開き、ホーム画面（[5. ホーム画面](05-home.md)）へ遷移する |
| `q` / `Esc` | 起動画面へ戻る |
| `Ctrl+C` | アプリを終了 |

## 読み込みと確定後の挙動

- ワークスペースの読み込みに失敗した場合は、空一覧の状態でエラー内容を通知行に表示します。
- `Enter` でワークスペースを選択するとホーム画面へ遷移します。このとき選択したワークスペースの
  最終利用時刻（`updated_at`）を現在時刻に更新するため、次回以降の一覧では最近開いたものが先頭に並びます。
- ホーム画面から戻る（`Esc`）とこの一覧へ復帰します。復帰時には今開いたワークスペースを
  一覧の先頭へ繰り上げて表示するため、再読み込みを待たずに「最近開いた順」が反映されます。
  アプリ終了（`Ctrl+C`）はそのまま上位へ伝播します。

> イベントループ `event.rs` に分離）に実装されています。一覧の元データ（`workspaces.json`）の
> 仕様は [../data/01-global.md](../data/01-global.md) を参照してください。
