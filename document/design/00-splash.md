# 0. 起動スプラッシュ（Splash）

> [画面設計トップ](README.md) ｜ 次へ → [1. 起動画面（Welcome）](01-welcome.md)

`usagi hop` 実行時、[1. 起動画面（Welcome）](01-welcome.md) に入る前に一度だけ再生される装飾的なオープニング。
ウェルカム画面と**同じうさぎ AA**を画面中央に表示し、少し遅れて `USAGI` タイトルがフェードインします。
全フレームの再生後、自動でウェルカムメニューへ切り替わります。うさぎ AA・タイトルはウェルカム画面と
共通なので、切り替え時に位置がガタつきません。代替スクリーンの初期化やマウスレポートの方針も
ウェルカム画面と共通です（[01-welcome.md](01-welcome.md) を参照）。

## レイアウト

うさぎ AA（3 行）とタイトルのブロックを端末の縦中央・横中央に配置します（幅・高さが取得できない環境では
80×24 をフォールバック値として使用）。うさぎ AA はウェルカム画面と同一で、再生中は動きません。タイトルの
行は最初から確保されており、フェードインで現れても周囲のレイアウトはずれません。

```text
┌───────────────────────────────────────────────────────┐
│                                                       │
│                       (\(\                            │  ┐
│                      (='-')                           │  ├ うさぎ AA（マゼンタ・太字／ウェルカムと共通）
│                      o(_(")(")                        │  ┘
│                                                       │
│                       USAGI                           │  ← タイトル（遅れてフェードイン → 緑・太字）
│                                                       │
└───────────────────────────────────────────────────────┘
```

## 構成要素

| 要素 | 内容 | スタイル |
|---|---|---|
| うさぎ AA | 3 行のアスキーアート `(\(\` / `(='-')` / `o(_(")(")`（ウェルカム画面と共通） | マゼンタ・太字（中央寄せ） |
| タイトル | `USAGI` | 緑系（中央寄せ）。フェードの最終段は緑・太字でウェルカム画面と一致 |

## アニメーション

うさぎ AA は再生中ずっと表示されたまま動かず、タイトルだけがアニメーションします。

- 最初の数フレームはうさぎ AA だけを表示し、タイトルは隠したままにします（少し遅らせる）。
- 遅延後、タイトルは緑の明度を段階的に上げながらフェードインし、最終段で緑・太字に達します。
  最終段はウェルカム画面のタイトルと同一なので、切り替え時に見た目が変わりません。
- フェードイン後は、ウェルカム画面へ切り替わるまで最終段のタイトルを数フレーム保持します。
- フレームは一定間隔（[ホーム/インストールと共通の描画間隔](../05-settings.md)）で自動的に進み、
  全フレームの再生後にウェルカム画面へ切り替わります。ごく短い演出です。

## 入力の扱い

スプラッシュは**入力を読み取りません**。自前のタイマーでフレームを進めるだけなので、再生中に押した
キーはバッファされ、そのまま次のウェルカムメニューに渡ります（タイプアヘッド）。ウェルカム画面が
ブロッキングで待ち受ける入力経路と競合しないため、再生中のキー入力が失われることはありません。

> 描画ロジックは `src/presentation/tui/splash/`（描画 `ui.rs`・イベントループ `event.rs`）、うさぎ AA と
> タイトル（フェード段階）の描画は共有ウィジェット `src/presentation/tui/widgets/`（ウェルカム画面と共通）に
> 実装されています。
